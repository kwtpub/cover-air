# Промпт для разработки дизайна в Figma AI

## Общее описание проекта

Разработай полный дизайн веб-сайта для платежного сервиса **COVER AIR** на базе **UDMPAY**. Это современная платформа для приема платежей через QR-коды и NFC технологию. Сервис предназначен для микробизнеса и банков, предлагает облачные решения для терминалов, касс, чеков и международных платежей.

## Брендинг и идентичность

**Название бренда:** COVER AIR / UDMPAY
**Слоган:** "Платежное решение 0₽ для клиентов"
**Стиль:** Современный, технологичный, премиальный, инновационный

## Цветовая палитра

### Основные цвета (бирюзовые/Tiffany)
- **Primary (Бирюзовый):** #0ABAB5
- **Primary Dark:** #089B96
- **Primary Light:** #00D4AA

### Светлые фоны
- **Белый:** #FFFFFF
- **Светло-серый темный:** #F5F5F5
- **Светло-серый светлый:** #F8FAFC
- **Средний серый:** #A0A0A0

### Темные цвета
- **Черный основной:** #000000
- **Черный темный:** #0D0D0D
- **Черный светлый:** #1A1A1A

### Градиенты
- **Основной градиент:** От бирюзового (#0ABAB5) к серому (#A0A0A0) под углом 135deg
- **Бирюзовый градиент:** От бирюзового к светлому бирюзовому
- **Радиальные градиенты:** Используются для создания глубины и акцентов

**Принцип использования цветов:** Бирюзовый (Tiffany) должен быть преобладающим цветом. Серый используется как вспомогательный. Черный для текста на светлых фонах.

## Типографика

### Заголовки
- **Шрифт:** Philo Logic Expanded (расширенный, декоративный шрифт)
- **Начертания:** Regular (400)
- **Размеры:** 
  - Hero заголовки: 48-72px
  - Основные заголовки секций: 32-48px (clamp для адаптивности)
  - Подзаголовки: 20-24px
- **Стиль:** Жирный (700), часто с акцентными элементами бирюзового цвета

### Основной текст
- **Шрифт:** Montserrat
- **Начертания:** 300, 400, 500, 600, 700, 800, 900
- **Размеры:**
  - Основной текст: 16px
  - Мелкий текст: 14px
  - Крупный текст: 18px
- **Высота строки:** 1.6-1.8
- **Цвет:** Черный (#000000) на светлых фонах, белый на темных фонах

### Акценты в тексте
- Бренд "COVER AIR" и "UDMPAY" часто выделяются бирюзовым цветом
- Используются жирные начертания для акцентов

## Структура сайта

## Дизайн-система и компоненты

### Кнопки
- **Основная кнопка:** Бирюзовый фон (#0ABAB5), белый текст, скругление 8-12px, padding 18px 40px
- **Черная кнопка:** Черный фон (#000000), белый текст
- **Hover эффекты:** Легкое поднятие (transform: translateY(-2px)), тень
- **Стиль:** Скругленные углы, современные, с плавными переходами

### Карточки
- **Фон:** Белый (#FFFFFF) или светлые оттенки
- **Границы:** 2px, бирюзовый цвет или серый
- **Скругление:** 12-16px
- **Тень:** Мягкие тени для глубины (0 4px 20px rgba)
- **Отступы:** 20-40px внутренние отступы

### Формы
- **Поля ввода:** Белый фон, скругление 12px, черный текст
- **Фокус:** Бирюзовое выделение (border или box-shadow)
- **Кнопки отправки:** Бирюзовые, жирный текст

### Иконки и визуальные элементы
- **Стиль:** Минималистичный, современный
- **Цвета:** Бирюзовый для акцентов, черный/серый для основного

## Особенности дизайна

### Градиенты
- Активно используются радиальные и линейные градиенты
- Бирюзовые градиенты для фонов секций
- Серые градиенты для создания глубины
- Градиенты создают плавные переходы между цветами

### Фоновые эффекты
- Размытые бирюзовые круги (blur эффекты) для атмосферы
- Радиальные градиенты для акцентов
- Текстуры и паттерны минимально

### Анимации и интерактивность
- Плавные переходы (transition: all 0.3s ease)
- Hover эффекты с поднятием элементов
- Smooth scroll
- Легкие трансформации

### Сетки и отступы
- Максимальная ширина контента: 1200-1400px
- Отступы по бокам: 40px (десктоп), 20px (мобильный)
- Отступы между секциями: 60-100px
- Gap в сетках: 20-40px

## Адаптивность

### Брейкпоинты
- **Desktop:** 1024px+
- **Tablet:** 768px - 1024px
- **Mobile:** до 768px
- **Small Mobile:** до 480px

### Принципы адаптива
- Гибкие сетки (grid-template-columns: repeat(3, 1fr) → 1fr)
- Clamp для размеров шрифтов
- Горизонтальные карусели на мобильных
- Уменьшенные отступы на мобильных

## Инструкции для Figma AI

1. **Создай систему компонентов:**
   - Кнопки (основная, черная, вторичная)
   - Карточки (базовая, с изображением, с иконкой)
   - Формы (поля ввода, селекты)
   - Типографика (заголовки H1-H4, body текст)
   - Цветовая палитра с переменными

2. **Создай макеты страниц:**
   - Главная страница со всеми секциями
   - Страница агента
   - Страница франшизы
   - Страница международных платежей

3. **Примени стили:**
   - Используй бирюзовый (#0ABAB5) как доминирующий цвет
   - Примени градиенты для фонов
   - Используй Philo Logic Expanded для заголовков
   - Montserrat для основного текста

4. **Создай варианты:**
   - Desktop версия
   - Tablet версия
   - Mobile версия

5. **Добавь интерактивность:**
   - Hover состояния для кнопок и карточек
   - Состояния форм (focus, error, disabled)

6. **Соблюдай принципы:**
   - Бирюзовый должен преобладать в дизайне
   - Современный, чистый стиль
   - Фокус на удобстве использования
   - Премиальный внешний вид

## Дополнительные заметки

- Дизайн должен передавать технологичность и инновационность
- Используй негативное пространство для читаемости
- Создай визуальную иерархию через размеры и цвета
- Убедись, что текст читаем на всех фонах
- Градиенты используются для создания глубины, не перегружай ими

---

**Цель:** Создать современный, профессиональный дизайн для платежного сервиса, который передает технологичность, надежность и инновационность бренда COVER AIR/UDMPAY, используя бирюзовую цветовую схему как основу визуальной идентичности.
